With Minimum Order Amount you will be able to set a minimum amount for users orders on Classipress theme. 

== Description ==

<strong><a href="https://www.appthemes.com/themes/classipress/?aid=26553" target="_blank">Classipress</a> theme is required for this plugin.</strong>

To use this plugin you should have installed at least one plugin that offers additional features for your users when posting ads like:
<br><a href="http://www.easycpmods.com/plugin-paid-ad-features/">Paid Ad Features</a>
<br><a href="http://www.easycpmods.com/plugin-paid-ad-extender/">Paid Ad Extender</a>

Plugin will not allow users to make purchases smaller then the amount you specify. Free ads are still allowed.  

It also works with other plugins.

== Installation ==

1. Extract the folder into your WordPress plugins directory
2. Enable the plugin
3. Config the plugin under Settings->Minimum Order Amount
